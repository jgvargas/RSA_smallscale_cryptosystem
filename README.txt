To start RSA encryption process you read two py files and one text file.

1.)rsa_main.py
2.)rsa_functions.py
3.)rsa_input.txt

When running rsa_main.py the program will automatically encrypt and decrypt the contexts of rsa_input.txt.

Key generation values will be printed out to console and unencrypted data shown in rsa_output.txt
